# Personal Information Manager

An interactive Command Line Interface (CLI) application built with Python. This project serves as a foundational implementation of core programming concepts, including static data assignment, dynamic user input collection, input validation, mathematical operations, and clean string formatting techniques.

---

## 🛠️ Technical Requirements Met

- **Data Structuring**: Utilizes variables to securely store baseline profile configurations using appropriate data types (Strings and Integers).
- **Dynamic Interaction**: Captures user preferences during runtime with interactive `input()` prompts.
- **Robust Validation**: Employs an error-handling loop to catch empty or invalid spaces, preventing program crashes.
- **Data Enhancements**: Features simple arithmetic calculations to convert a static age in years to total months.
- **Polished Output**: Formats outputs gracefully using f-strings and Python string methods (`.upper()`, `.title()`, and `.strip()`).
