# Smruti
"A beginner-friendly Python CLI application that stores, processes, and displays personal user data, featuring input validation, dynamic user interaction, and age-to-months calculations."




#student grade calculator
Conditional Logic (if/elif/else): Used inside get_grade() to dynamically bracket user averages into functional performance categories.

Lists & Dictionaries: Using dictionaries ({}) allows you to group different data types (Strings, Lists of Floats, calculated averages) cleanly per individual student, creating a master list array (student_records) that scales seamlessly.

While Loops & Defending Code (try/except): The get_valid_int() and get_valid_float() functions act as structural gates. If someone types "eighty" or accidentally punches a negative value, the while loop combined with try-except cleanly absorbs the error without crashing the script.

Iterative Lists (for loops): Processes each individual score array to determine statistics such as max(), min(), and class-wide performance totals.
